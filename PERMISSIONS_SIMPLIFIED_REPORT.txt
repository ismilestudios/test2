iSmile Scheduler permissions update

- User roles simplified to Admin and Photographer.
- Admin page Users & Permissions section now shows a simple key: Photographer users cannot remove events, restore removed events, see the Admin page, or merge schools.
- Older Scheduler / Viewer-Photographer role values normalize to Photographer in the UI.
- Photographer users no longer see the Admin tab.
- Photographer users no longer see Remove Event, Restore Removed Event, or Merge School actions.
- Admin users retain full access.
